SQLAlchemy 仓储教程
==============================


在本教程中,我们将逐步介绍对简单数据库关系建模的过程,并演示 Litestar 的仓储模块如何使数据库工作变得轻而易举。让我们开始吧!


.. admonition:: 本教程适合谁?
    :class: info

    本教程涵盖 Litestar 的 SQLAlchemy 仓储。它假设你理解 Litestar 的关键概念,以及一定程度的 SQLAlchemy 熟悉度。
